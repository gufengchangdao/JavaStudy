泛型尖括号细节
============

```java
import java.util.ArrayList;

class Test {
	public static void main(String[] args) {
		//<>表示携带泛型信息，只是省略了而已，不含<>的则不携带，默认为边界，这里是Object类型
		ArrayList<String> list1 = new ArrayList<>();
		ArrayList list2 = new ArrayList();
	}

	// Collections一段代码，在new的时候没有看到泛型信息，但是确确实实携带给构造器了的
	// 在调用checkedList()时，第二个表明为Dog类型，第一个尖括号内没有写类型，要写只能写Dog，甚至写Object都会报错
	List<Dog> list5 = Collections.checkedList(new ArrayList<>(), Dog.class);

	public static <E> List<E> checkedList(List<E> list, Class<E> type) {
		return (list instanceof RandomAccess ?
				new CheckedRandomAccessList<>(list, type) :
				new CheckedList<>(list, type));
	}
}
```
