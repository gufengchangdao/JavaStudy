Stream操作分组
==============

1. 终端操作
    + forEach、forEachOrdered
    + toArray
    + reduce
    + collect
    + min、max
    + count
    + anyMatch、allMatch、noneMatch
    + findFirst、findAny
2. 中间操作(返回新流)
    + filter
    + map、mapToInt、mapToLong、mapToDouble
    + flatMap、flatMapToInt、flatMapToLong
    + distinct
    + sorted
    + peek
    + limit
    + skip
    + takeWhile
    + dropWhile
